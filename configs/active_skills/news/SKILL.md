# 新闻查询技能

Agent 可使用此技能获取实时新闻和资讯。

## 能力说明

- **web_search**：通过 DuckDuckGo 搜索实时新闻
- **http_request**：访问新闻 API 或 RSS 源

## 使用场景

- 查询今日头条新闻
- 搜索特定主题的最新报道
- 获取科技/财经等领域资讯

## 使用建议

1. 使用 web_search 时提供明确的关键词
2. 可指定时间范围如「今日」「本周」
3. 新闻类查询优先使用 web_search 获取实时结果
